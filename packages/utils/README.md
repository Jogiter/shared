# shared

shared utils

## install

```sh
$ yarn add @np/utils
```

## develop

```sh
# dev
yarn start

# commit(better git-commit)
yarn commit

# build
yarn build

# test
yarn test

# lint
yarn lint

# package size analyze
yarn analyze

# generate doc
yarn doc

# publish to npm(better npm-publish)
yarn release
```
